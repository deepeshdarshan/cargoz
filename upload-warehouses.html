Upload Warehouses to Firestore

Paste your warehouse JSON array in the text area below and click Start Upload.

⚠ Firestore Field Names (case-sensitive)

  • name — Warehouse ID
  • area — Sub-location
  • location — Main location
  • cargoType — Cargo type
  • Type of Warehouse — Warehouse type
  • storageType — Storage type
  • capacity — Capacity (number)
  • unit — Unit (Sqft / Sqm / Pallet)
  • starGrading — Star rating (number)
  • temperature — Temperature control
  • safetyAndSecurity — Safety info
  • otherFacilities — Facilities
  • imageUrl — Image path
  • rate — Monthly rate (number)

⚠ Field names are case-sensitive. Use the exact keys listed above in your JSON.
